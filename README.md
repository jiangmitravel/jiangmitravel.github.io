### 2. Beijing Food Menu Image Processor

A batch image processing tool designed for food photography and restaurant menu optimization, running entirely in your browser.

- **GitHub**: [beijingfoodmenu-image-tools](https://github.com/jiangmitravel/beijingfoodmenu-image-tools)
- **Live Demo**: [https://jiangmitravel.github.io/beijingfoodmenu-image-tools/](https://jiangmitravel.github.io/beijingfoodmenu-image-tools/)
- **Features**:
  - Batch image processing
  - Quality adjustment and resizing
  - Format conversion (JPEG/PNG/WebP)
  - Privacy protection (EXIF metadata removal)
  - ZIP archive download
